## Getting Started

```bash
$ gem install bundler

# Create Gemfile
$ bundle init
# Writing new Gemfile to /path/to/Gemfile

$ bundle install
# Bundle complete!

$ bundle check
# The Gemfile's dependencies are satisfied

# Show Gems in the current Gemfile
$ bundle list
# Gems included by the bundle:
#   * bundler (1.11.2)
```
