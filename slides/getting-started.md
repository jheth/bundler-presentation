## Quick Start

```bash
$ gem install bundler

# Create Gemfile
$ bundle init
# Writing new Gemfile to /path/to/Gemfile

$ vim Gemfile
# Add gem 'sinatra'

$ bundle install
# Bundle complete!

$ bundle check
# The Gemfile's dependencies are satisfied

$ bundle list
Gems included by the bundle:
  * bundler (1.11.2)
  * rack (1.6.4)
  * rack-protection (1.5.3)
  * sinatra (1.4.7)
  * tilt (2.0.2)
```
